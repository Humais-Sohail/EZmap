<h1 align="center">EZmap</h1>
<p>Ezmap is a CLI based tool that simplifies the nmap scanning. It is fast, easier to use and uses nmap under the hood. To use this you must have nmap installed on your system otherwise it will simply not work. This tool was developed in C++.</p>

<img src="/screenshots/1.png">
<hr>
<h1>Working</h1>
<p>EZmap will only work if the nmap is installed on your system.</p><br>
<p>Check nmap https://github.com/nmap/nmap </p>
<p>User must start or use EZmap with <b>'sudo'</b> otherwise some scans may not be executed properly.</p>
<hr>
<p>First clone the repository</p>
<pre>git clone https://Humais-Sohail/EZmap</pre>
<p>Get into the folder</p>
<pre>cd EZmap</pre>
<p>Give permissions and start the tool</p>
<pre>chmod +x EZmap<br>sudo ./EZmap</pre>
<p>Give the targets and select the scan type you want to use... :)</p>
